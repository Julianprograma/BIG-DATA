Contextualizacion del problema

Este proyecto simula un sistema inteligente basado en IoT para detectar vehículos robados en Bogotá mediante el uso de cámaras de fotodetección utilizando bases de datos públicas como el RUNT, la ANSV y registros de hurto vehicular. Actualmente, estas cámaras se usan con fines sancionatorios, pero no existe una integración automatizada que permita identificar robos en tiempo real. La propuesta busca potenciar su uso preventivo, integrando procesos ETL y técnicas de Big Data para limpiar, unificar y analizar la información, generando alertas automáticas cuando un vehículo sospechoso es detectado. Los resultados se visualizarán en Power BI, lo que facilitará la interpretación de datos y la toma de decisiones por parte de las autoridades. A nivel estratégico, el proyecto busca mejorar la seguridad ciudadana, fomentando ciudades más seguras e innovadoras a través del uso inteligente de la infraestructura tecnológica existente.

Proceso IoT

En este proyecto, el proceso de IoT se simula a través de una red de cámaras de fotodetección que generan 500.000 reportes de paso vehicular, simulando sensores reales. Estos datos fueron consultados y extraídos en formato CSV para su análisis. Posteriormente, se realizó una limpieza en Power BI, eliminando campos irrelevantes como direcciones específicas, priorizando atributos clave como la localidad. La información tratada permitió estructurar una base de datos optimizada. A partir de ella, se generaron reportes analíticos que reflejan patrones de circulación y presencia de vehículos robados. Este flujo imita cómo un sistema IoT real captaría, procesaría y visualizaría datos en tiempo real. El resultado es un sistema inteligente de monitoreo urbano dirigido a la seguridad de los ciudadanos
